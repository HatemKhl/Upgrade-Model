# Broadband Upgrade Financial Model

A React and TypeScript dashboard for comparing broadband network upgrade scenarios from an existing HFC DOCSIS 3.1 low-split plant.

## Scenarios

- DOCSIS 3.1 High-Split
- DOCSIS 4.0 HFC
- FTTH XGS-PON

## Run Locally

```bash
npm install
npm run dev
```

Then open the local Vite URL printed by the terminal, usually `http://127.0.0.1:5173/`.

## What Is Included

- Editable global and scenario assumptions
- Annual free cash flow model
- NPV, ROI, IRR, payback, and cumulative cash flow KPIs
- Dashboard charts with Recharts
- Scenario comparison view
- Plain-English formulas and methodology
- Approval request form with frozen submitted snapshots and mock approval actions
